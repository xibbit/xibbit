The Django development server uses static/index.html instead of / (or /index.html) to deliver static content.

You can look at the actual static/ serving Django source code at a path like:

/Library/Frameworks/Python.framework/Versions/3.10/lib/python3.10/site-packages/django/views/static.py

and see that there is no way to serve index.html by default; it is hardcoded to directory_index.html.

Click static/index.html to start.

Click http://localhost:8000/admin/login/?next=/admin/ to see the Django admin panel
